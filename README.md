# Pets Rescue

**Pets Rescue** é um sistema desenvolvido em **Java** como projeto acadêmico para a faculdade.
O objetivo do projeto é simular um sistema simples de gerenciamento de **adoção e resgate de animais**, permitindo registrar usuários, cadastrar pets e organizar pedidos de resgate.

A proposta do sistema é demonstrar conceitos fundamentais de **programação orientada a objetos**, como criação de classes, uso de métodos, entrada de dados com `Scanner` e organização de um sistema baseado em **menu interativo no terminal**.

## Funcionalidades do sistema

O sistema possui um menu principal que permite:

* Cadastro de usuários (doadores ou adotantes)
* Cadastro de pets com identificação própria
* Registro de adoção de pets com escolha da data de retirada
* Solicitação de resgate de animais em situação de risco
* Listagem de pets cadastrados no sistema
* Listagem de pedidos de resgate registrados

## Objetivo acadêmico

Este projeto foi desenvolvido como parte de uma atividade da disciplina de **Programação**, com o objetivo de aplicar na prática conceitos como:

* Estruturas de classes e objetos
* Manipulação de dados em Java
* Uso de menus interativos
* Organização básica de um sistema de gerenciamento

## Tecnologias utilizadas

* **Java**
* **Terminal / Console**
* Estruturas básicas da linguagem (classes, métodos e arrays)
